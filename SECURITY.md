# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please report it privately:

- Email: security@visafact.org

Include:
- A clear description of the issue
- Steps to reproduce
- Impact assessment (if known)

We will acknowledge receipt within 3 business days and provide a status update within 10 business days.

## Supported Versions

We support the `main` branch and the latest release snapshot.

## Scope

Security issues include:
- Data integrity flaws in evidence validation
- Unsafe URL handling in the UI
- CI/CD pipeline vulnerabilities

## Disclosure

We follow coordinated disclosure and will credit reporters who request attribution.
